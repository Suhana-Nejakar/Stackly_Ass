# Student Marks Analyzer using 1D and 2D Arrays (Core Java)

## 1. Overview

This application demonstrates **core Java array concepts** using a real‑world scenario: analyzing student marks.

You will learn:

* How to use **1D arrays** for names and totals
* How to use **2D arrays** to store tabular data (students × subjects)
* How to process arrays using **loops only** (no collections, no streams)
* How to separate **utility logic** and **business logic**

## 3. Data Model Explanation

### 3.1 Students

Stored in a **1D String array**:

```java
String[] studentNames = {"Rahul", "Anjali", "Vikram", "Sneha", "Kiran"};
```

Each index represents one student.

---

### 3.2 Subjects

Stored in another **1D String array**:

```java
String[] subjectNames = {"Maths", "Science", "English", "CS"};
```

Each index represents one subject.

---

### 3.3 Marks (Core Concept)

Stored in a **2D int array**:

```java
int[][] marks = {
  {78, 85, 90, 88},
  {92, 80, 76, 89},
  {65, 70, 68, 72},
  {55, 60, 58, 62},
  {88, 75, 82, 91}
};
```

Interpretation:

* `marks[i]` → all subject marks of student `i`
* `marks[i][j]` → marks of student `i` in subject `j`

## 4. ArrayUtils.java (Utility Class)

### Purpose

Provides **reusable static methods** that operate only on arrays.

### Why static?

* No object needed
* Easy reuse across classes

### Implemented Methods

| Method                 | Purpose           |
| ---------------------- | ----------------- |
| `sum(int[])`           | Total of elements |
| `max(int[])`           | Highest value     |
| `min(int[])`           | Lowest value      |
| `average(int[])`       | Mean value        |
| `indexOf(int[], int)`  | Linear search     |
| `copy(int[])`          | Safe array copy   |
| `reverse(int[])`       | In‑place reverse  |
| `printMatrix(int[][])` | Print 2D array    |

All methods use **loops only**.

## 6. Mandatory Methods – Step‑by‑Step Logic

### 6.1 Print Marks Table

Uses nested loops:

* Outer loop → students (rows)
* Inner loop → subjects (columns)

---

### 6.2 Total Marks per Student

For each student row:

```text
sum of marks[i][0..n]
```

Uses `ArrayUtils.sum()`.

---

### 6.3 Average Marks per Student

```text
total marks / number of subjects
```

---

### 6.4 Total Marks per Subject

For each subject column:

```text
sum of marks[0..n][j]
```

---

### 6.5 Average Marks per Subject

```text
total subject marks / number of students
```

---

### 6.6 Overall Topper

Steps:

1. Get total marks array
2. Find index of maximum value
3. Return student name at that index

---

### 6.7 Subject‑wise Toppers

For each subject column:

* Compare all students’ marks
* Track highest mark & student index

---

### 6.8 Grades per Student

Based on **average marks**:

| Range  | Grade |
| ------ | ----- |
| 90–100 | A     |
| 80–89  | B     |
| 70–79  | C     |
| 60–69  | D     |
| <60    | F     |

---

### 6.9 Students Failed in Any Subject

Logic:

* If **any mark < passMark**, student fails
* Two‑pass approach:

  1. Count failures
  2. Create exact‑size array

---

### 6.10 Class Average

```text
(total of all marks) / (students × subjects)
```

---

### 6.11 Get Marks for a Student

Steps:

1. Search student name
2. Return **copy** of marks row
3. Return `null` if not found

---

### 6.12 Subject Index Finder

Simple linear search in `subjectNames[]`.

---

### 6.13 Pass Percentage for a Subject

```text
(passCount / totalStudents) × 100
```
